如何在Ubuntu 20.04上安装配置Redis

Redis 是一个开源的在内存存储键值对数据的存储程序。它可以被用作数据库，缓存，信息暂存，并且支持各种数据结构，例如：字符串，哈希值，列表，集合等等。 Redis 通过 Redis Sentinel 和 Redis 集群中多个 Redis 节点的自动分块处理，提供了高可用性。

这篇指南描述了如何在 Ubuntu 20.04 上安装和配置 Redis。
